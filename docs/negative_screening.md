# 投資対象外銘柄（ネガティブ・スクリーニング）定義書

## 業種基準

1. 物理的・構造的制約セクター（資本集約型）
物理的インフラの維持と減価償却費が利益を圧迫する業種。非線形な成長が困難。
• 電力・ガス業
• 陸運業・海運業・空運業
• 倉庫・運輸関連業

2. コモディティ・市況依存セクター
製品の差別化が困難で、価格決定権を自社で持てない業種。
• 鉄鋼・非鉄金属
• パルプ・紙
• 繊維製品
• 鉱業

3. 労働集約・カオス心理セクター
属人性が高く、スケールメリットが効きにくい、または消費者心理の予測が困難な業種。
• 小売業
• サービス業（低付加価値型）
• 銀行業・証券・商品先物取引業

4. 低資本効率・規制セクター
資本効率（ROE）が構造的に低く、市場の期待値が低い業種。
• 建設業
• 水産・農林業

---

## 数理的基準（アルゴリズム）

以下のいずれかに該当する場合、投資対象から排除する。
1. 高い限界費用: 売上1単位あたりの物理的コストが線形に増大するか。
2. 低い研究開発比率: 売上高R&D比率が低く、技術的参入障壁が構築されていないか。
3. 物理的在庫の重層性: 在庫回転率が低く、キャッシュフローが物理的実体に拘束されているか。
